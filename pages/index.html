Kid-Friendly API

Cuick was carefully designed to have the simplest API imaginable, with a minimal learning curve and familiar conventions. You can even choose between JSX-like template literals, regular JSX, or handlebars. Check out the docs to learn more.

Interactive Prototyping

When you nest a component in a <cuick-story> element, it automatically builds a playground with controls for the nested component's props and CSS variables. Besides letting you test and debug your component as you write it, this is a truly effortless way to generate documentation, user stories, or markup for your projects.

Context API

Cuick's Context API makes it incredibly easy to synchronize state between a parent and multiple child components. This takes a lot of the pain out of developing otherwise complex features like app-wide i18n. See the i18n component page for a working demo.
