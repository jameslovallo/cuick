Cuick

Cuick is your Custom-Element UI Crafting Kit.

Get Started Components Templates

Features

  • Simple object-oriented API
  • Interactive prototyping
  • JSX-like templates (or handlebars)
  • Reactive props and state
  • Easy to use Context API
  • Extensive component library
  • App starter templates

Interactive Prototyping

As you write a new component, wrap it in a <cuick-story> element. It creates an interactive playground by generating controls for your component's props and any CSS variables defined in the :host selector. This is also a fantastic way to document your component, create user stories, and generate markup for your sites and apps.
